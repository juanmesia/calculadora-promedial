# Calculadora Promedial — PWA para GitHub Pages

Versión corregida para celular.

## Cambios aplicados

- Se cambió `index(3).html` a `index.html`.
- Se corrigió la compatibilidad móvil del JavaScript.
- Se eliminaron funciones que pueden fallar en navegadores móviles antiguos.
- Los campos numéricos ahora aceptan punto o coma decimal.
- Se actualizó el service worker a `calculadora-promedial-v4` para obligar a limpiar caché anterior.
- Se agregaron iconos PNG para instalación como PWA.

## Cómo publicar

Sube todos los archivos de esta carpeta a la raíz del repositorio de GitHub.
Luego activa GitHub Pages en:

Settings > Pages > Deploy from a branch > main > /root

## Si en celular sigue abriendo la versión vieja

1. Borra la app de la pantalla de inicio.
2. Abre el enlace en el navegador.
3. Refresca la página 2 veces.
4. Instala nuevamente la app.
