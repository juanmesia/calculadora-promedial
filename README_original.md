# Calculadora Promedial

Calculadora movil para estimar una tasa de proteccion en VES/USD cuando hay inflacion o fluctuacion del dolar.

## Que calcula

- Variacion entre una tasa inicial y una tasa actual.
- Promedio diario de aumento.
- Tasa proyectada al final del periodo.
- Tasa sugerida para mantener precios durante varios dias.
- Precio sugerido en VES segun valor base en USD, margen y colchon adicional.

## Como usarla

Abre `index.html` en el navegador o publica el proyecto con GitHub Pages.
